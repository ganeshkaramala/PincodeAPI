Welcome to PINCODE API

Below are the available API's

  • Information of a Pin code : /v1/pinocde/:pincode/info
  • Distance near by a Pin code : /v1/pinocde/:pincode/distance?parameter1=value1,..
    Parameters
    • nearby = district,subregion,region
    • radius = a value in k.m
    • limit = 0-100
